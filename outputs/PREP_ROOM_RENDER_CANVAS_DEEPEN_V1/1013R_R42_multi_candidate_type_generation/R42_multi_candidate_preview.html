R42 多类型候选预览

所有内容仅用于模型质量沙盒，不保存、不导出、不覆盖原教案。

teaching_process_cleanup

status: fallback

source: fallback_quality_sandbox

1. 导入：出示色彩渐变图片，引导学生观察并思考'颜色是怎样变化的'。
2. 讲解：结合图片讲解明度渐变的基本概念，举例说明从深到浅或从浅到深的变化规律。
3. 探究：组织学生两人一组，比较同一色相不同明度的色卡，找出明暗差异。
4. 实践：学生尝试在调色盘中加入白色或黑色进行调色，亲手制作一组明度渐变颜色，并完成一幅渐变色彩小作品。
5. 展示与评价：小组内互评，全班展示优秀作品，教师小结。

小教建议: 这是沙盒候选，只用于质量判断和教师二次编辑，不写入正式备课本。

缺口: 教师确认
